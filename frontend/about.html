About MyAuthApp

This is a full-stack authentication app built with Node.js, Express, Postgres (on Render), and vanilla JavaScript. It features secure email verification using Nodemailer (Gmail SMTP), Google OAuth via Passport, JWT authentication, profile management with image uploads (Multer), light/dark theme toggling, and an admin dashboard for user CRUD operations (like direct SQL edits).

Key Technologies:

  • Backend: Node.js, Express, pg (Postgres), bcryptjs, jsonwebtoken, nodemailer, passport-google-oauth20, multer
  • Frontend: HTML5, CSS3 (with custom properties for themes), Vanilla JS (fetch API)
  • Database: Postgres on Render (tables: users, temp_verifications, home_content)
  • Deployment: Render (Web Service for backend, Static Site for frontend)

Security: Password hashing (bcrypt 12 rounds), JWT (1h expiry), Helmet (headers), CORS (frontend only), unique constraints (email/username).

Developed for secure user management. No React used (vanilla JS as specified), but extensible.
